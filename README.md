# Linux Basic Server

This project documents the process of setting up an Ubuntu Server from scratch for web hosting and remote administration.

## Objectives

- Install Ubuntu Server
- Configure SSH
- Configure UFW Firewall
- Install Git
- Install Nginx
- Basic Linux Administration

## Environment

| Component | Version |
|-----------|---------|
| Ubuntu Server | 24.04 LTS |
| Nginx | Latest |
| Git | Latest |
